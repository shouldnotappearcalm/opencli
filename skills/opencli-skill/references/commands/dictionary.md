# dictionary

## Commands

### examples
- Purpose: Get example sentences for a given word
- Args: None
- Usage: `opencli dictionary examples [options] -f json`

### search
- Purpose: Search dictionary definitions, parts of speech, and pronunciations
- Args: None
- Usage: `opencli dictionary search [options] -f json`

### synonyms
- Purpose: Find synonyms for a given word
- Args: None
- Usage: `opencli dictionary synonyms [options] -f json`
