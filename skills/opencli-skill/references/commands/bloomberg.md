# bloomberg

## Commands

### businessweek
- 用途：Bloomberg Businessweek top stories (RSS)
- 参数：
  - `limit`（可选）：类型 int；默认值 1；Number of feed items to return (max 20)
- 用法：`opencli bloomberg businessweek [options] -f json`
